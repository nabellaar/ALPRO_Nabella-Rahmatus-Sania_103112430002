Program Menentukan Nilai Maksimum dan Minimum

Kamus:
- a, b, c, d : integer
- max : integer
- min : integer

Algoritma:
1. Input nilai a, b, c, dan d
2. max = a
3. min = a
4. Jika b > max maka:
    max = b
5. Jika c > max maka:
    max = c
6. Jika d > max maka:
    max = d
7. Jika b < min maka:
    min = b
8. Jika c < min maka:
    min = c
9. Jika d < min maka:
    min = d
10. Tampilkan nilai max dan min

End Program
